# Machine-Learning-Model-Predict-Evaluate
Iris ML Project
Overview
This project demonstrates basic data preprocessing and exploratory analysis using the Iris dataset. The dataset is cleaned, inspected, and prepared for machine learning tasks.

Technologies Used
Python
Pandas
Scikit-learn
Jupyter Notebook
Workflow
Load the Iris dataset
Explore dataset structure
Check for missing values and duplicates
Encode categorical labels
Prepare features and target variables for machine learning
Dataset
The Iris dataset contains 150 samples of iris flowers with four features:

Sepal Length
Sepal Width
Petal Length
Petal Width
Target classes:

Setosa
Versicolor
Virginica
Outcome
The dataset is successfully preprocessed and ready for use in classification models.
